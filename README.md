# ⬛ NeoWallet - Single Product Funnel

A high-converting, dark-mode e-commerce landing page designed for dropshipping and single-product physical goods.

## ✨ Features

- **Conversion Rate Optimization (CRO):** Sticky "Buy Now" bar, urgency countdown timer, and trust badges.
- **Modern UI/UX:** Sleek dark mode with neon accents, smooth hover states, and mobile-first responsive design.
- **Clean Architecture:** HTML, CSS, and JavaScript completely separated into individual files for easy client handoff.
- **Interactive Elements:** Dynamic countdown timer, toast notifications for cart actions, and mobile hamburger menu.

## 🛠 Tech Stack

- **Markup:** Semantic HTML5
- **Styling:** Vanilla CSS (CSS Variables for easy theme changes)
- **Logic:** Vanilla JavaScript (No jQuery or heavy frameworks)
